# 🎓 GPA Calculator (4.3 Scale)

A simple, interactive web-based GPA calculator built with **HTML, CSS, and JavaScript**.
This tool allows users to input courses, assign grades and credits, and instantly calculate their GPA on a **4.3 scale**.

---

## ✨ Features

* ➕ Add and remove courses dynamically
* 📊 Supports standard **4.3 GPA scale** (A+ to F)
* 🧮 Automatic GPA calculation based on credits
* 📈 Displays:

  * Grade-to-percentage mapping
  * Per-course GPA breakdown
  * Overall GPA
* 💡 Clean and responsive UI

## 📚 GPA Scale Used

| Grade | GPA | Percentage |
| ----- | --- | ---------- |
| A+    | 4.3 | 90–100%    |

| A-    | 3.7 | 80–84%     |
| B+    | 3.3 | 77–79%     |
| B     | 3.0 | 73–76%     |
| B-    | 2.7 | 70–72%     |
| C+    | 2.3 | 67–69%     |
| C     | 2.0 | 63–66%     |
| C-    | 1.7 | 60–62%     |
| D+    | 1.3 | 57–59%     |

| D-    | 0.7 | 50–52%     |
| F     | 0.0 | 0–49%      |

## 🧠 How It Works

1. Add courses using the **“+ Add Course”** button
2. Enter:

   * Course name
   * Credit value
   * Grade
3. Click **“Calculate GPA”**
4. View:

   * Individual course results
   * Final GPA

The GPA is calculated using:

```text
GPA = Total (Grade Points × Credits) / Total Credits
```
